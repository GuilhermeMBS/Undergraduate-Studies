# Undergraduate Studies & Lab Exercises

This repository is a collection of some of the most relevant coursework, lab assignments, and self-study exercises developed during my **Computer Engineering** degree at **PUC-Rio**.

## 📌 Overview

The goal of this repository is to track my learning progress and archive small-scale implementations from my university classes and personal study sessions. It serves as a personal "notebook" for my academic evolution.

## 📂 Content

* **Lab Assignments:** Code developed for weekly university classes.
* **Exercises:** Small scripts and snippets used to practice specific concepts.
* **Small Projects:** Minor academic implementations, such as a **Huffman Compressor** (Data Structures) and a **BigInt** library in C (Systems Programming).

**P.S.** The files are organized by semester and subject to reflect the curriculum's timeline.
